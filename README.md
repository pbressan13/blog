PreRequisites

* Ruby version:  3.4.2 
  - Rails 8.0.1
  - PostgreSQL
  - `curl` (for testing POST requests)
  - `git` (for cloning the repository)
  
* Clone repository 

   ```gh repo clone pbressan13/blog
   cd ratings-app
   ```

* Install dependencies
    ```bundle install```
    
* Database creation
    ```rails db:create```

* Run migrations:
    ```rails db:migrate```

* Seed the database (optional):
    ```rails db:seed```
